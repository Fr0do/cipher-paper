\section{Experiments}
\label{sec:experiments}

\subsection{Models}

We benchmark the following agents, all using the same three-stage pipeline
(Section~\ref{sec:task}) with per-model VLM backends:

\begin{itemize}
  \item \textbf{OCR-only} (baseline): easyocr on sampled frames; no VLM.
  \item \textbf{Claude Haiku}: \texttt{claude-haiku-4-5} via Anthropic API.
  \item \textbf{GPT-4o-mini}: \texttt{gpt-4o-mini} via OpenAI API.
  \item \textbf{Gemini Flash}: \texttt{gemini-2.0-flash} via Google API.
  \item \textbf{Qwen2-VL-7B}: open-weight, served locally on 4$\times$H100.
  \item \textbf{Qwen2-VL-72B}: open-weight, served locally on 4$\times$H100.
\end{itemize}
